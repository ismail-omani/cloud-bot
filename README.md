# cloud-bot
Source code of https://t.me/ismail_cloud_bot

Installation:
You need python3 and git installed.

Download installer.sh(Linux) or installer.bat(Windows)

For Linux:
```bash
bash installer.sh
```
For windows:
Double-click on installer.bat or
```bash
installer.bat
```

If something stuck, press enter.
After installing change "YOUR_TOKEN" in "API_TOKEN = 'YOUR_TOKEN'" in bot.py to your bot token, then run "python bot.py" being in directory cloud-bot
